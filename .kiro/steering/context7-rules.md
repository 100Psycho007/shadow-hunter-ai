# Context7 Auto-Usage Rules

## Documentation and Code Examples

When the user asks about:
- Code examples for any library or framework
- How to use a specific API or package
- Setup or configuration steps
- Library documentation
- Framework best practices
- Implementation patterns

Always automatically use the `get-library-docs` tool from Context7 to fetch up-to-date documentation.

## Examples of when to use Context7:
- "How do I create a Next.js API route?" → Use Context7
- "Show me React hooks examples" → Use Context7  
- "Configure Tailwind in my project" → Use Context7
- "What's the latest Supabase auth setup?" → Use Context7

## When NOT to use Context7:
- General programming questions
- Debugging specific code
- Project-specific logic
- Questions about user's own code
